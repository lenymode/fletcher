J. Fletcher Art — Website Redesign Submission

Live Netlify Project
https://fletcherart.netlify.app

Submission Overview
This package presents a premium, contemporary e-commerce redesign for J. Fletcher Art. The concept is focused on photographic impact, refined typography, natural collection browsing, simplified A3/A4/A5 print purchasing, and conversion-friendly product pages.

The theme structure includes seven homepage directions plus shared commercial pages for Shop, Product Detail, Cart, Checkout, Collections, About, and Journal. This gives the client a flexible premium-theme style presentation: multiple homepage choices with consistent buying and content templates.

Key Features
- Seven homepage visual directions.
- Shop page with filtering, sorting, page size control, and pagination.
- Product detail page with image slider and related works.
- A3, A4, and A5 print sizing with uniform pricing.
- Cart and checkout templates ready for payment integration.
- Detailed About, Collections, and Journal pages.
- Sticky navigation, responsive layouts, and clean CSS structure.
