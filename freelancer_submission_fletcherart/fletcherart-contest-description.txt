Live Netlify Project

Submission Overview
This concept presents a refined, contemporary e-commerce redesign for J. Fletcher Art, built around photographic impact, clean typography, clear product navigation, and a calmer buying journey. The design direction focuses on large editorial imagery, generous spacing, premium art-gallery styling, and direct paths toward product discovery and Add to Cart actions.

The build includes seven homepage directions so the client can choose the strongest visual route while keeping the core shop, product, cart, checkout, about, collections, and journal pages consistent across the theme. This is similar to a premium marketplace/themeforest-style theme structure: multiple homepage concepts, shared commercial templates, and reusable design components.

Key Features Delivered
- Seven unique homepage concepts.
- E-commerce friendly product grid and product detail templates.
- A3, A4, and A5 print size options with uniform pricing.
- Working shop filters, sorting, page size controls, and pagination.
- Product detail image slider for multiple product images.
- Related works section with See All link.
- Sticky navigation and visual homepage switcher.
- Cart and checkout template structure ready for payment integration.
- About, Collections, and Journal pages for richer brand storytelling.
- Mobile-first responsive layout across desktop, tablet, and mobile.
- Editable CSS architecture split into main styles and responsive styles.

Design Direction
The proposed storefront uses a gallery-inspired visual language: large photographic compositions, restrained colours, refined serif headings, simple sans-serif UI text, and clean product cards. The goal is to make the artwork feel premium immediately while keeping the shopping path simple and intuitive.

Sizing and Product Offer
The product structure follows the simplified client requirement: no paper-option complexity, with uniform pricing across three standard print sizes: A3, A4, and A5.
